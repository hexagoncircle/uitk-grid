UITK Grid

The UI Toolkit Grid is the underlying structure that aids designers and developers in the construction of web app layouts. Review this documentation and try it out in the embedded Codepen below.

You can add as many columns as you want to the grid. Columns will span across a 12-column layout and then wrap onto a new row once they exceed this limit. The following example shows how HTML attributes are used to acheive a flexible grid-based layout.

Column sizing

Set column sizes using the span attribute. Use span-[size] to set a new column size at a particular breakpoint.

Push Columns

Set push-[size] to push a column over grid spaces. Use push-reset-[size] to remove the push attribute at a particular breakpoint.

Grid Gap Sizing

The gap between columns can be resized using the gap attribute. Breakpoint gap sizing is available using gap-[size]. Available size values are [loosen, default, tighten, none].

Nested Grids

Simply add a new grid inside of a column to nest a grid inside a grid.

Vertical Alignment

Align elements vertically using the align-y attribute. Set it to [flex-start, center, flex-end] to align elements to the top, middle or bottom of their container

Horizontal Alignment

Align elements horizontally using the align-x attribute. Set it to [flex-start, center, flex-end] to align elements to the left, center, or right of their container.

Distribution

Distribute elements horizontally using the align-x attribute. Set it to [space-around, space-between] to align elements to assign spacing around or between each column.

Attach Grid

A grid can have another grid element attached to it using the attach-grid attribute. This will supress the bottom margin on this particular grid element to create a seamless collection of grids.

Reordering

Reorder elements using the order attribute.
